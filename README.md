# 🖱️ Cursor Tracker Box

A simple and interactive JavaScript project that tracks the movement of the mouse cursor on the screen and moves a **red rectangular box** according to the cursor position.

This project is built using **HTML, CSS, and JavaScript DOM concepts** to understand real-time mouse events and dynamic UI updates.

---

## 🚀 Features

- Real-time mouse movement tracking
- Red rectangular box follows cursor movement
- Smooth UI interaction
- Beginner-friendly DOM project
- Responsive browser behavior

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (DOM)

---

## 📚 Concepts Used

- Mouse Events
- `mousemove` Event Listener
- DOM Manipulation
- Dynamic Positioning
- CSS Positioning
- Event Handling
